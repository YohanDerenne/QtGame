# Controls
| KEY         | Action         |
|-------------|----------------|
| R           | Respawn Player |
| ESCAPE      | Pause          |
| LEFT ARROW  | Move left      |
| RIGHT ARROW | Move right     |
| UP ARROW    | Jump           |
| F11         | Full Screen    |
| SPACE       | Attack         |

# Consumable Objects
Heart : give one health point
Mask : Immune for a short time
Hydroalcoholic Gel : Allow you to attack with a gel projectile for a short time
