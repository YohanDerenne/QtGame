# Controls
| KEY         | Action         |
|-------------|----------------|
| R           | Respawn Player |
| LEFT ARROW  | Move left      |
| RIGHT ARROW | Move right     |
| UP ARROW    | Jump           |
| F11         | Full Screen    |
| SPACE       | Attack         |
